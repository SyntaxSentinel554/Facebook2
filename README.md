## 一、账户创建与配置
### 1. 注册四要素
- **基础信息架构**  
  主页名 = 主营产品关键词 + 服务类型（例：LEDLighting_Wholesale）  
  用户名 = 品牌缩写 + 行业后缀（例：SunTech_LED）  

- **验证体系搭建**  
  ① 美国实体SIM卡接收验证码  
  ② PayPal企业账户绑定支付  
  ③ 域名邮箱匹配企业官网
